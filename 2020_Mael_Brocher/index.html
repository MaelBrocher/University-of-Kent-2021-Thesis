These directories contain everything needed to compile and use the project.

The first one, ./Source/, is the JavaScript/Python and HTML + CSS. It's a website deploy locally on localhost:5000.
Follow the README.md to run the project.

The second directory is data I used to test and improve the project. It came from Gary Read that started the project in 2016.

And finally the last one contains screenshots of use cases of the project.

README.md

Make sure you have python3 installed, if so just type "sudo ./init.sh"
or type the commands in the second section.

If not :

    sudo apt-get install python3

Then do :

    sudo apt-get install python3-pip
    pip install --upgrade pip
    pip install setuptools wheel spacy flask langid
    python -m spacy download en_core_web_sm
    python -m spacy download fr_core_news_sm
    flask run 

And open your browser at localhost:5000
